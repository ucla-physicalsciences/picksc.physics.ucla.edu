Coarray Fortran enables a programmer to write parallel programs using a Partitioned Global Address Space (PGAS) scheme. It is part of the Fortran2008 standard and provides an alternative to the dominant Message-Passing Interface (MPI) paradigm.

The code ppic2_caf is provided by Alessandro Fanfarillo and Damian Rouson, and is a translation of the MPI code ppic2.
